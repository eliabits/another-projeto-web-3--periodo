# Pikachu Catalogando

Projeto desenvolvido para a disciplina de Desenvolvimento Web onde mostro os conhecimentos adquiridos sobre grid, flexbox, fetch, tailwind css e consumo de API.
o projeto é bem simples, com página única onde podemos procurar os pokemons pelo nome ou número na barra de busca. 

## Tecnologias

- HTML
- Tailwind CSS
- JavaScript
- Fetch API
(não utilizei nenhum documento css pois o tailwind ta ai pra isso)

## API utilizada

PokeAPI (api de lista de pokemons, o código acessa a api para solicitar o pokemon pesquisado, caso não haja, retorna um aviso de pokemon não encontrado)
[🔗 PokeAPI](https://pokeapi.co/api/v2/pokemon)

## Funcionalidades

- Buscar Pokémon pelo nome
- Buscar Pokémon pelo número
- Exibir imagem
- Exibir tipos
- Exibir altura
- Exibir peso
- Exibir habilidades
 (a cor do card depende do tipo do pokemon, se for fogo, água, grama... cada cor é relacionada ao tipo principal)
