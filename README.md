# Pikachu Catalogando

Projeto desenvolvido para a disciplina de Desenvolvimento Web onde mostro os conhecimentos adquiridos sobre grid, flexbox, fetch, tailwind css e consumo de API.

## Tecnologias

- HTML
- Tailwind CSS
- JavaScript
- Fetch API
(não utilizei nenhum documento css pois o tailwind ta ai pra isso)

## API utilizada

PokeAPI
https://pokeapi.co/api/v2/pokemon

## Funcionalidades

- Buscar Pokémon pelo nome
- Buscar Pokémon pelo número
- Exibir imagem
- Exibir tipos
- Exibir altura
- Exibir peso
- Exibir habilidades
